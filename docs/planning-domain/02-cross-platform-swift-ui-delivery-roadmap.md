# 02 Cross-Platform Swift UI Delivery Roadmap

## Purpose

This document is the overview for the cross-platform Swift UI delivery planning cluster.

## Planning Cluster

The planning cluster is now split into two main tracks:

1. shared-core-to-shell delivery sequencing as the implementation workstream
2. platform-shell decision and proof as the alternative shell-evaluation workstream

## Workstream Relationships

1. [[06-shared-core-to-shell-delivery-sequence]] is the primary implementation workstream.
2. [[07-platform-shell-decision-and-proof-workstream]] is the shell-evaluation workstream that helps choose or validate Linux-first, Apple-first, or abstraction-first proof paths.
3. The UI planning workstreams depend on design documents rather than replacing them.

## Stable Planning Position

1. shared Swift core first
2. Linux UI proof second
3. Apple shell expansion after the shared core is proven reusable
